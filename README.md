## 📌 Project Explanation

SavePlate is a smart platform developed to reduce food wastage by connecting food donors (such as restaurants, hostels, and individuals) with receivers (NGOs, needy people, etc.) in real-time.

The system allows users to easily donate surplus food through a web or mobile interface. The donated food is then listed in the system along with location and details. Nearby receivers can view available food and request it instantly.

The platform uses location-based tracking to ensure quick and efficient redistribution of food. This helps in minimizing food waste and ensures that excess food reaches people in need instead of being discarded.

Overall, SavePlate aims to create a sustainable and efficient food-sharing ecosystem using modern web and mobile technologies.
